# VidyaMitra – The Intelligent Career Agent 🤖💼

**VidyaMitra** is a modern full-stack AI-powered career guidance platform designed to help students and fresh graduates navigate the job market.

## 🌟 Project Overview
The system integrates Artificial Intelligence to provide personalized assistance through resume evaluation, mock interviews, and career roadmapping. 

### 🏗️ Architecture
- **Frontend:** HTML5, CSS3 (Glassmorphism UI), Vanilla JavaScript.
- **Backend:** Python FastAPI (Asynchronous API).
- **Logic Engine:** Keyword Mapping, Skill Gap Analysis, and Mock Interview Simulation.

## 🚀 Key Scenarios Covered
1. **Scenario 1: AI Resume Evaluation:** Analyzes technical skills to provide an ATS score.
2. **Scenario 2: Mock Interview Simulator:** Interactive quiz with AI feedback on confidence and accuracy.
3. **Scenario 3: Upskilling Planner:** Generates a 4-phase roadmap for career transitions (e.g., shifting to Data Science).

## 🛠️ How to Run Locally
1. Clone the repository.
2. Start the Backend: `python server.py` (Runs on Port 8010).
3. Start the Frontend: Open `index.html` with Live Server.
